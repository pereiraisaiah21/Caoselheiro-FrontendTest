## ⚙️ Instruções

Clone o repoitório:
https://github.com/pereiraisaiah21/teste-front-end

Após isto, via terminal:
cd caoselheiro

Atualize as dependencias:
npm install

E em seguida, para executar rode:
npm start

Com isto o projeto será aberto em seu navegador.
